Step 1: Convert Your Image

First, you'll need to convert your image to a base64 string. You can do this using:

  • Our online converter tool
  • Browser DevTools Console
  • Command line tools
# Using command line (macOS/Linux)
base64 image.png | pbcopy

Step 2: Format the Markdown

Use this syntax to embed your base64 image in Markdown:

![Alt text](data:image/png;base64,YOUR_BASE64_STRING)

Replace YOUR_BASE64_STRING with your actual base64-encoded image data.

Step 3: Best Practices

Keep these tips in mind:

  • Use base64 for small images (under 50KB) to keep your Markdown files manageable
  • Consider using standard image hosting for larger files
  • Always include meaningful alt text for accessibility
  • Compress images before converting to base64 to reduce file size
